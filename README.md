# DisasterManager

 
**Developed by:** Znady Rime & Bouklhatem A bdelilah 

##  Project Overview
**DisasterManager** is a natural disaster management system designed to monitor, detect, and respond to emergencies efficiently.  
It provides **real-time alerts** and **map-based visualization** to help authorities and users stay informed during critical situations.

##  Features
- Real-time disaster notifications  
- Interactive map for disaster tracking  
- Emergency monitoring and alert system  
- Data management and reporting dashboard  

##  Technologies Used
- **Java**  
- **Spring Boot**  
- **MySQL**  
- **Maps API**  

##  Purpose
To provide an intelligent and responsive platform for **emergency management**, improving coordination and minimizing disaster impacts.
